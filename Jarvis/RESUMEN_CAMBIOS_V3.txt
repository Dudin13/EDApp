🐍 BACKEND: labeller_app.py
------------------------------------------

[SEGURIDAD - ALTA PRIORIDAD]
✅ Path traversal corregido en save_labels()
   → Antes: un filename como "../../etc/passwd" podía salirse del dataset
   → Ahora: función safe_resolve() valida que el path final esté dentro de DATASET_ROOT

✅ CORS restringido a localhost
   → Antes: CORS(app) aceptaba cualquier origen
   → Ahora: solo permite http://localhost:* y http://127.0.0.1:*

✅ Whitelist de subsets válidos
   → Añadida constante VALID_SUBSETS = {"train", "val", "test"}
   → Todos los endpoints validan el subset antes de continuar

[ROBUSTEZ - ALTA PRIORIDAD]
✅ Manejo de imagen nula en segment_image() y predict()
   → Antes: img.shape[:2] petaba con AttributeError si cv2 no podía leer el fichero
   → Ahora: se comprueba if img is None y se devuelve error 500

✅ Validación de coordenadas del punto en segment_image()
   → Se verifica que point tenga 'x' e 'y', y que estén en rango [0,1]

✅ Validación de labels en save_labels()
   → Se verifica que labels no sea None y sea una lista

✅ Creación automática del directorio de labels si no existe
   → lab_path.parent.mkdir(parents=True, exist_ok=True)

[RENDIMIENTO]
✅ SAM con carga lazy (get_sam_model())
   → Antes: SAM se cargaba al arrancar Flask aunque no se usara (ocupaba VRAM innecesariamente)
   → Ahora: se carga solo en el primer request a /api/segment

✅ Fichero temporal único por request en predict()
   → Antes: tmp_pred.jpg era un fichero global compartido (riesgo de corrupción con concurrencia)
   → Ahora: se genera un nombre único con uuid4() en el directorio temporal del sistema
   → El fichero se elimina automáticamente tras enviarlo (after_this_request cleanup)

[PORTABILIDAD]
✅ Paths via variable de entorno APPED_ROOT
   → Antes: "C:/apped" hardcodeado directamente
   → Ahora: BASE = Path(os.environ.get("APPED_ROOT", "C:/apped"))
   → Permite cambiar el path base sin tocar el código
